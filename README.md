# typedJS
Example of a bare-bones apps using Flow and TypeScript


## Dependenies
- Java 8
- node/npm


## Getting Started
### Start the server
- `cd server/`
- `./gradlew build`
- `./gradlew bootrun`
- app will run on localhost:8080 by default

### start a client
- choose an app and `cd` into the directory
- `npm install`
- `npm start`
- app will run on locahost:8000 by default
